                        # 流程控制

                        # 真或假

                        Java中的真或假分别用，trueorfalse来表示，但是在Java中不允许用1或0来表示，尽管其他语言是有这种用法

                        # if-else语句

                        流程控制最基本的形式，其语法如下

                        if(布尔类型表达式){
    语句;
}else if(布尔类型表达式){
    语句;
}else{
    语句;
}

                        布尔类型表达式说明其计算结果是boolean类型，其中else if和else可以省略

                        # 迭代语句

                        迭代语句包含三种类型,只有当循环控制条件是false迭代才会停止，只要是true一直会循环

                        • while
                        • do while
                        • for

                        # while和do while

                        语法结构为

                        while(布尔类型表达式){
    语句
}

do 
    语句
while(布尔表达式);

                        do while区别于while，其永远会执行一次

                        # for

                        语法结构为

                        for(初始表达式; 布尔表达式; 步进表达式){
    语句
}

                        上述的三个表达式和语句其执行顺序为初始表达式->布尔表达式->语句->步进表达式,例如

                        public static void main(String[] args) {
    for (int i = 0; i < 5; i++) {
        System.out.println(i);
        System.out.println("---");
    }
}

输出
0
---
1
---
2
---
3
---
4
---

                        逗号操作符，在for循环中可以使用逗号分割不同的变量，并且只能用于初始化表达式和步进表达式位置，如下所示

                        public static void main(String[] args) {
        for (int i = 0,j = i + 1; i < 5; i++,j++) {
            System.out.println(i);
            System.out.println(j);
            System.out.println("---");
        }
    }

                        # for in

                        for in该语法在Java5中出现,又叫做for-each,其是用来操作数组和与集合，语法格式如下

                        for(初始表达式:迭代目标){
    语句
}

例如
public static void main(String[] args) {
        int[] numbers = {1,2,3,4,5};
        for (int n : numbers) {
            System.out.println(n);
        }
    }

                        # return

                        return关键字作用有如下两方面，其一作为方法的返回值，其二，退出当前方法，并返回值

                        如下所示

                        static int test(int n1, int n2){
    if(n1 > n2){
        System.out.println(n1);
        return n1;
    }
    if(n1 < n2){
        System.out.println(n2);
        return n2;
    }
    System.out.println(0);
    return  0;
}
public static void main(String[] args) {
    test(1,2);
}

输出2

                        上述例子中，方法的返回类型是int，因此必须显式指明返回值，如果是void类型，其实也有return，只不过是隐式指明的

                        # break和return

                        break和return是用于迭代循环中的关键字，其中break表示跳出循环体，continue表示结束本次循环，开始下一次循环

                        如下演示了for中break和return运用

                        for(int i = 0; i < 10; i++) { // [1]
    if(i == 9) break; // 跳出循环
    if(i % 2 != 0) continue; // 下一次循环
    System.out.print(i + " ");
}

0 2 4 6 8

                        上述例子，第i=9的时候循环跳出终止了;当i是奇数的时候，结束当前循环，不再执行下面语句，跳入下一次循环

                        下面是一些无限循环的一些小写法

                        while(true){

}
for(;;){

}

                        # label

                        goto是跳转语句，在其他语言中存在，但在Java中并没有实现，只是作为保留关键字

                        尽管，Java没有goto跳转，但是有一个label标签，配合break和continue关键字，可以看到类似goto的功能,格式如下

                        label1:
outer-iteration { 
  inner-iteration {
  // ...
  break; // [1] 
  // ...
  continue; // [2] 
  // ...
  continue label1; // [3] 
  // ...
  break label1; // [4] 
  } 
}

                        outer-iteration和inner-iteration分别表示外层循环和内层循环，其中continue label1;停止当前这一次的内部循环和外部循环，然后重新进入标签处进行下一次循环;break label1则表示终止标签所指的循环

                        下面是一个例子

                        public static void main(String[] args) {
outer:
for(int i = 0; i < 5; i++) { // [1]
    System.out.println("----i:"+i);
    inner:
    for(int j = 1; j < 4; j++){
        if (j == 2){
            continue outer;
        }
        System.out.println("j:"+j);
    }
}
}

                        上面例子，j永远递增不到4，这是因为当j=2就到达outer所在位置的循环，并进入下一次循环

                        # switch

                        switch是指选择语句，其语法格式如下

                        switch(产生整数的表达式){
    case 整数:
        语句 break;
    ...
    default:
        语句;
}

                        该语法说明，会根据整数的表达式所产生的数值进行匹配下面，若匹配到进入执行相应语句，若一直没匹配到那么default后面的语句会执行;其中如果带break表示匹配到了就不再继续往下寻找，若没有break尽管已经匹配到还是会往下寻找

                        Java7新增字符串匹配,如下所示

                        public class StringSwitch {
  public static void main(String[] args) {
    String color = "red";
    // 老的方式: 使用 if-then 判断
    if("red".equals(color)) {
      System.out.println("RED");
    } else if("green".equals(color)) {
      System.out.println("GREEN");
    } else if("blue".equals(color)) {
      System.out.println("BLUE");
    } else if("yellow".equals(color)) {
      System.out.println("YELLOW");
    } else {
      System.out.println("Unknown");
    }
    // 新的方法: 字符串搭配 switch
    switch(color) {
      case "red":
        System.out.println("RED");
        break;
      case "green":
        System.out.println("GREEN");
        break;
      case "blue":
        System.out.println("BLUE");
        break;
      case "yellow":
        System.out.println("YELLOW");
        break;
      default:
        System.out.println("Unknown");
        break;
    }
  }
}
