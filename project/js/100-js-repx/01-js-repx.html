                    # JS正则表达式

                    正则表达式（英语：Regular Expression，在代码中常简写为regex、regexp或RE）使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式。

                    搜索模式可用于文本搜索和文本替换。

                    • 正则表达式是一个搜索模式

                    # 基本语法

                    var patt = /表达式内容/i;

                    i 表示不区分大小，/ /里是表达式主体内容

                    # 字符串方法

                    • search方法，传入正则表达式搜索指定内容或传入指定内容搜索，并返回起始位置
                    • replace方法，传入正则表达式进行替换或传入指定内容替换，并返回替换后的内容

                    search方法，如下

                         var str = "Visit ismeoh";
    var n = str.search(/ismeoh/i);
    console.log(n);// 返回起始6

                    search方法，直接传入字符串

                    var str = "Visit ismeoh";
var n1 = str.search("ismeoh");
console.log(n1); // 6

                    replace方法类似，如下

                            // replace用法
        var str2 = "Visit ismeoh";
        var n2 = str2.replace(/visit/i,"Hello");
        console.log(n2); //
        var n3 = str.replace("ismeoh","Hello");
        console.log(n3); //

                    # 正则表达式修饰符

                    var patt = /表达式内容/i;

                    如上，i所在位置就是修饰符，修饰符除了i还有如下

                    • i 不区分大小写
                    • g 执行全局匹配，即查找所有匹配内容，而不是在找到第一个匹配后停止
                    • m 执行多行匹配

                    # 正则表达式模式

                    方括号，用于查找范围内的值

                    • [abc]查找方括号之间的任何字符。
                    • [0-9]查找任何从 0 至 9 的数字。
                    • (x|y)查找任何以 | 分隔的选项

                    元符号拥有特殊含义的符号

                    \d  查找数字。
\s  查找空白字符。
\b  匹配单词边界。
\uxxxx  查找以十六进制数 xxxx 规定的 Unicode 字符。

                    量词

                    n+ 匹配任何包含至少一个 n 的字符串。
n* 匹配任何包含零个或多个 n 的字符串。
n? 匹配任何包含零个或一个 n 的字符串

                    # RegExp对象

                    在 JavaScript 中，RegExp 对象是一个预定义了属性和方法的正则表达式对象

                    # 使用test()

                    test() 方法是一个正则表达式方法。

                    test() 方法用于检测一个字符串是否匹配某个模式，如果字符串中含有匹配的文本，则返回 true，否则返回 false。

                    # 使用 exec()

                    exec() 方法是一个正则表达式方法。

                    exec() 方法用于检索字符串中的正则表达式的匹配。

                    该函数返回一个数组，其中存放匹配的结果。如果未找到匹配，则返回值为 null。
