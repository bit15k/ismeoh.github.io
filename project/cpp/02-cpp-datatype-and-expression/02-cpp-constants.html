                # C++常量

                常量是指不可改变的量，主要分为如下

                • 数值型常量，10
                • 字符型常量，包含在` `里的量

                # 数值常量

                数值常量也是分类型的，主要包含如下

                • 整型常量

                # 整型常量

                对应数据类型中int，比如10至于整型常量的范围不需要掌握，用到查询即可

                整型常量的表达方法，主要有如下三种

                • 十进制数
                • 八进制数
                • 十六进制数

                十进制数字需要在数字尾部加l或L，例如10L
                八进制数字需要在开头加0,八进制的020代表十进制16
                十六进制在数字的开头加0x或0X，例如0x20即32

                # 浮点型常量

                表达方式主要有两种

                • 十进制小数
                • 指数形式

                十进制小数形式
                例如0.12345，主要包含整数部分和小数部分，浮点数常量根据数据类型主要分为两种

                • 双精度浮点类型常量
                • 单精度浮点类型常量

                双精度浮点类型常量，一般默认是双精度，例如0.123，其一般占内存8字节
                单精度浮点类型常量，在数字后面加f或F

                指数形式表达
                主要形式0.123e10，具体如下

                # 字符常量

                主要分为如下两种

                • 普通字符常量
                • 转义字符常量

                # 普通字符常量

                ` `包裹的内容，例如`a`，需要注意的是

                • 只能包含一个字符
                • 只占1字节
                • 区分大小写

                # 转义字符常量

                例如\n不表示字母n，而表示换行符

                常见的转义字符查询

                注：转义字符尽管包含两个字符，但是仍然占1个字节，编译系统会将ASCII码进行转换

                字符数据在计算机存储是以ASCII码存储的，所以字符数据可以赋值给整型变量，反之也可将>>整型数据赋值给字符变量

                常用ASCII码表

                # 字符串常量

                使用" "包裹起来的内容，例如"abc"
                字符串常量，占4个字节

                字符串常量在内存中的表示

                其中\0不是字符串的一部分，因此当输出的时候不会输出

                cout<<"abc"<<endl //abc

                char类型不能使用" "，因为只能包含一个字符

                char c;
c = "abc"  //❌

                # 符号常量

                cpp可以使用#define来定义一个符号常量，例如

                #define PRICE 30
int main(){
    int num.total;
    num = 10;
    total = num *PRICE;
    cout<<"total="<<total<<endl;
    return 0;
}
