                              # 基本句型

                              单句主要分为5种基本句型

                              • 主语动词
                              • 主动宾补
                              • 主动宾宾
                              • 主动宾
                              • 主动宾补

                              其中上面的动词大陆叫谓语动词，上面的主语S,动词v,宾语o,表语c来表示

                              主部和述部

                              • 主部以主语为核心，告诉的读者主角
                              • 述部以先动词为核心，告诉主角怎么样

                              限定动词与非限定动词

                              • 限定动词就是根据时态变化的动词，即传统语法动词
                              • 非限定动词是动状词，动状词是指现在分词、过去分词、动词不定式等等

                              基本句型的分类依据

                              根据以下动词类型分为5种

                              根据及物分

                              • 及物动词 对应3种，即主动宾、主动双宾、主动宾补、主动补
                              • 不及物动词 对应1种，即主动

                              根据动词意思分

                              • 完全动词 对应主动、主动宾、主动双宾、主动宾补
                              • 不完全动词 对应1种，即主动补

                              # 种类

                              # 主动词

                              主语+动词，主要包含如下内容

                              • 理解名词的作用
                              • 理解名词种不及物动词的作用
                              • 理解介词片语及其作用
                              • 理解词组意义
                              • 理解介词和介副词，以及介副词的作用
                              • 理解动词词组以及时间副词

                              例1

                              • 狗冲着邮差叫 The dog barked at the maillman

                              句子解释

                              • 狗当作主语，这里是名词
                              • 叫当作动词，该动词是不及物动词是指不需要加宾语就能完成这个动作
                              • 介词片语at the mailman是指邮差做at的宾语，这里介词片语修饰动词

                              例2

                              • 今天早上我起的很晚,I got up rather late this morning

                              句子解释

                              • 我是代词做主语
                              • 起床got up是一个词组是指两或以上单词合在一起构成一个意思，这里get单独使用没有起床的意思，必须和up合起来用才有，这里的up是指介副词是指看起来是介词但是后面没有宾语，当作副词用。词组动词也是当作动词使用，只不过这里是不及物动词。
                              • rather late表示时间副词词组，修饰动词

                              # 主动宾

                              主语+动词+宾语，主要包括

                              • 理解及物动词及其宾语

                              例1

                              • 狗去咬邮差,The dog bit the mailman

                              句子解释

                              • 这里的咬是指及物动词，后面必须跟着这个动作承受者,这个动作的承受者叫做宾语，这里宾语是指邮差，宾语可以名词类,是指具有名词作用的词，比如代词、名词词组、名词从句等等
                              • 一个动词可以既有不及物用法也有及物用法

                              # 主动宾宾

                              主语+动词+宾语+宾语，主要包括

                              • 理解双宾
                              • 理解授予动词
                              • 掌握双宾的顺序变换

                              主动宾宾是指动词后面跟两个宾语，是及物动词范畴，通常这里的及物动词带有授予的意思，因此叫做授予动词，第一个宾语叫做授予对象，第二个宾语叫做授予东西

                              例1

                              • 这个人的老婆给了他一罐蜂蜜,The man's wife gave him a jar of honey

                              句子解释

                              • 此句子的him表示授予对象，a jar of honey表示授予东西

                              该句型一般的顺序应该是主语+动词+授予对象+授予东西，若要调换东西和对象的顺序，需要加介词主语+动词+东西+介词+对象形式

                              # 主动补

                              主语+动词+补语，主要包括

                              • 理解不完全动词
                              • 理解主语补语
                              • 掌握判断不完全动词的方法

                              不完全动词是指，动词的意思不完全，后面需要补语，如is，主语做什么一般是由动词来说明，但是若只翻译成是，无法表述完全，因此后移到补语来说明主语,也叫作主语补语

                              例1

                              • 这只狗是一只斗牛犬,The dog is a bulldog

                              句子解释

                              • 这里的动词 是 意思不完全，只是起了衔接的作用，说明此狗是什么，需要后面的补语，通常这种动词叫做系动词，后面叙述部分叫做补语

                              • 判断动词是否是不完全动词的简便方法，将其换成是，若意思差不多，一般就是不完全动词

                              # 主动宾宾补

                              主语+动词+宾语+宾语补语，主要包括

                              • 理解宾语补语
                              • 掌握判断主语动词宾语补语句型判断一般方法

                              例1

                              • 他老婆叫他Honey,His wife call him Honey

                              句子解释

                              • 叫的后面出现了两个部分，第一个部分him叫做宾语，第二部分叫做宾语补语

                              判断该句型简便方法在句子的宾语和补语之间加一个是，能够构成主动补就是主动宾宾补

                              # 补语的词类

                              补语的词类，主要包括如下

                              • 掌握主语和宾语的词性
                              • 掌握补语的词性
                              • 掌握补语的分类
                              • 理解补语和修饰的区别
                              • 主语和宾语必须是名词类
                              • 主语或宾语的补足语使用的词类相同，可以是名词类、形容词、时间副词、介副词、地方副词5种

                              # 名词类补语

                              名词做主语补语

                              • John是我的好朋友John is my best friend,这里的my best friend名词类做主语的补语

                              名词做宾语补语

                              • 我认为join是我的好朋友,I consider John my best friend,这里my best friend做宾语的补足语

                              # 形容词补语

                              形容词做主语补语

                              • John看起来很高兴,John looks happy

                              形容词做宾语补语

                              • 好消息让John很高兴,The good news makes John happy

                              # 时间副词做补语

                              时间副词做主语补语

                              • 决赛在今天晚上,The final game is tonight

                              时间副词做宾语补语

                              • 主办方把决赛安排在今晚,The sponsors set the game tonight

                              注：tonight词性是时间副词，注意区分补语和修饰的关系，这里是做补语补充说明，不是指修饰名词翻译成...的，翻译顺序也可知

                              # 地方副词做补语

                              地方副词做主语补语

                              • 钥匙在那里,The key is there

                              地方副词宾语补语

                              • 我把药匙留着那里，I left the key there

                              # 介副词做补语

                              介副词做主语补语

                              • 飞机着陆了，The plane is down

                              介副词做宾语补语

                              • 驾驶员带领飞机着陆了，The pilot brought the plane down

                              注：这里的down是指介副词，表示类似介词形式，但是后面并不不会跟宾语

                              关于补语和修饰的说明
                              我们一般说补语是指，补充说明主语或补语，是按照句子成分来说。修饰一般说是翻译成的，是按照词性说明。如上，时间副词、地方副词、介副词都可以当做补语，尤其是做宾语的补语，宾语是名词类，不能说成修饰，因为副词不能修饰名词，而是要说成做补语，这样才符合一般逻辑。

                              # 基本句型区分

                              主要包括

                              • 掌握区分主动补和主动宾区分的方法
                              • 掌握区分主动宾补和主动双宾区分的方法

                              是指根据主语、动词、宾语、宾语补语、主语补语，主要元素数目来区分

                              • 两个主要元素，主动词
                              • 三个主要元素，主动宾，主动表
                              • 四个主要元素，主动双宾，主动宾补

                              # 两个元素句型

                              主语+动词，句尾出现副词

                              那只猫叫的很可怜，The cat meowed pitifully,这里的副词是修饰动词，非主要成分

                              # 三个元素句型

                              主动宾

                              • 猫捉了一只老鼠，The cat caught amouse

                              主动补

                              • 这只猫看起来很优雅，The cat looked elegant

                              注：区分主动宾和主动补，就是在宾语后加个is试一下，如果意思差不多就是

                              # 四个元素句型

                              主动双宾

                              • 这个人喂猫吃罐头，The man fed the cat canned food

                              主动宾补

                              • 此人称这只猫叫胖子，The man called the cat 'Fatty'

                              注：区分主动双宾和主动宾补，如同主动宾和主动补的区分
