    # 名词词组

    # 名词词组基本构成有哪些？

    名词词组的三个部分限定符+形容词+名词,三个部分任何一部分都有可能省略，当然最好在写作中要同时出现。一个好学生，a good teacher，我们通常所说的名词其实是名词词组中省略了限定符和形容词部分，因此以后遇到名词一律按照名词词组看待，这样能解释很多内容

    # 名词词组中的名词

    # 名词按照数目分有哪几种？

    可数名词和不可数名词，可数名词是指限定符可以使用a、an等表述数目的词修饰，不可数名词采用零冠词策略，即前面没有冠词修饰，但是可以使用其他的非表示数目的词修饰，例如人称代词my等词

    # 可数名词有哪几种？

    第一种，固定形状的物品的是可数名词，例如a boy
    第二种，度量衡单位是可数名词，例如a pound一磅

    # 不可数名词有哪几种？

    第一种，物质名词
    什么是？
    即一些没有形状的物质，例如water
    物质名词是否可以用冠词？
    不可以，因为其不可数，所以采用零冠词策略
    物质名词如何进行计量？
    使用其对应的单位，例如一张纸，a sheet of paper
    什么叫物质名词普通化？
    即如果一个物质名词前面采用了冠词等数目词修饰产生了可数与不可数的形态，那么其不再当做物质名词了，而是普通名词，例如a paper不是指一张纸，而是指一份报告

    第二种，抽象名词
    什么是抽象名词？
    是指一些抽象的概念，例如智慧wisdom
    抽象名词是否使用冠词？
    采用零冠词策略
    抽象名词的普通化？
    例如a real beauty

    第三种，动名词
    什么是动名词？
    即在动词的后面加上ing，即ving，其代表一些动作或事情，例如Smoking 、swimming
    动名词是否采用冠词？
    不采用

    第四种，专有名词
    是什么专有名词？ 指一些地名、人名，其具备唯一性，即世界上只有一个，例如London，以大写字母开头的形式出现
    专有名词是否加限定符？
    不加，因为其已经具备唯一性了
    专有名词出现复数的情况是什么意思？ 其出现复数表示其已变成可数名词的形式，例如the Toms表示汤姆一家

    # 复合名词

    # 复合名词是什么？

    复合名词是指，在限定符+形容词+名词这个结构中形容词位置使用名词，用名词代替形容词的功用，这种叫做复合名词

    # 复合名词有几种写法？

    第一种，两个名词合在一起无空隙，例如the dishwasher
    第二种，两个名词加-链接，例如the dish-washer
    第三种，两个名词分开，例如the dish washer

    # 复合名词中间的名词单复数有什么要求？

    中间的形容词位置的名词使用单数，即便是单词原本就是复数型，也必须使用单数，除非使用单数形式导致意思发生改变才会使用，例如trousers本身就是复数单词，其变为复合名词your trouser pocket成了单数，在例如a clothes hanger表示衣架，如果使用a cloth hanger意思变成了布架
    当然一些惯用的复数形式要采用复数，例如a sports car跑车

    # 复合名词复数有什么要求？

    复合名词的复数形式写在最后一个名词上，至于中间的名词单复数根据实际情况。除了一些特殊的情况，例如由介副词组成的名词可能会加在介副词尾部，例如100 push-ups是指100个俯卧撑

    # 名词词组中的副词作用？

    这一部分是指名词词组的中间部分的形容词，还可以加副词修饰，例如the rather old jacket里面的rather是副词，用来修饰形容词表示加强语气，这一部分有单独的副词章节说明

    # 名词词组中的形容词的要求？

    名词词组中的形容词可以有多个，例如a juicy Japanese apple

    # 名词词组的省略

    # 名词词组中可以省略哪些内容？

    三个部分任何一部分都有可能省略，其中限定符部分的省略通常叫做零冠词，因此可以说除了限定符部分其他部分都有可能省略

    # 名词词组的中的名词省略？

    省略名词要不影响对其的理解，省略名词其实就是构成了the+adj=名词，所以说至于后面使用is or are 看还原后情况，例如the old表示老人，原本the old man

    # 限定词

    # 限定词按照其位置分为哪几种？

    分为前位、中位、后位，当然这几个位置的限定词是可以全都出现，也可以出现一个或两个都可，例如all these many brave people

    # 前位限定符一般有哪些词？

    主要包含如下词all both half one-third double twice、three time、uch等等，例如all the girls构成的前位+中位+名词

    # 后位限定符一般有哪些词？

    主要包含数字one two、序数first、表示多少的many much little few等，例如my three childen构成的中位+后位+名词

    # 中位限定符一般有哪些词？

    中位限定符是最常见的，如冠词a、an、the、零冠词、指示限定词this、that、these、those、不定限定词some\any\either、否定限定词no\neither、全称限定词every\each、所有格my\your\Tom's等,例如a book

    # 限定符具体使用的要求？

    可以全部出现，也可以只出现一个，但是要注意出现多个，要按照前中后的顺序排好，例如my all boys这是错误的，其成了中位+后位了；
    同时每个位置的限定符只能出现一次，例如a this boy是错误的，其出现了两个中位限定符

    # 名词词组里面的名词词组

    # 名词词组里面的名词词组是什么？

    是指在两个位置即限定符位置和形容词位置，可以使用名词词组代替

    # 限定符位置的名词词组可以放什么词？

    例如可以放所有格形式，如a man's hornor,其中a man's是一个名词词组放在了限定符位置

    # 名词位置的名词词组

    例如a unbelievable 200 billion dollars,其要求将其名词位置里的名词词组200 billion dollars看做一个整体，所以使用了a是单数

    # 名词词组中形容词探讨

    # 名词词组中形容词的位置是否可以放名词词组？

    不能，其夹在中间，塞不进了

    # 名词词组中的形容词部分是否可以放名词？

    可以，放入名词当做形容词使用，例如a flower shop一个花店，像这种叫做复合名词，其实在复合名词已经说过，只不过当时只是形式，没说当做形容词使用

    # 名词位置如果非要放名词词组如何进行改造？

    需要使用连字符进行改造，例如a three-story house一个三层楼，其中三层楼是名词词组，需要注意的是如果改成连字符原来的复数形式要变成单数形式，例如three stories三层楼

    # 名词与限定符的一致性是指？

    单复数一致如many books，同时不能用修饰非可数名词的限定符修饰可数名词例如less jobs错误，应该用fewer jobs，反过来同理不能用修饰可数名词的限定符修饰不可数名词，如many wisdom许多智慧，错误

    # 限定符之冠词

    # 如何简单的判断用冠词？

    如果一个名词可以用一个东西，那么可以用，例如a apple，a water错误

    # 零冠词是什么？

    零冠词是为了占位的而引出的虚拟概念，其实如果采用零冠词表示在其限定符位置是空的，例如new book前面看似没有限定符，但为了完整性我们说前面采用了零冠词

    零冠词后面的名词可能是什么形式 单数或复数，或不可数名词

    # 定冠词

    定冠词是怎么来的？
    定冠词是由that\those弱化而来，换句话说可以相互代替，其中其代表具有指示性，因此the也带有指示性

    如何判断是否使用定冠词？
    替换成that/those看其是否说的通即看齐是否具有指示性，例如John is a very tall player, but he’s not the tallest player on the team

    最高级与比较级中的定冠词如何用？

    我们老是说最高级加定冠词，但是有特殊情况。需要这样理解即要是有指示性那就用the没有指示性不用the，而不是去套什么规则，例如用the的情况，the most expensive books in my library具有强烈的指示性意味，与之相反的Most books in my library are science fiction.只是讲了大多数无指示性意味，但是这里most仍然表示最高级

    例如Mangoes are most expensive in early summer, when they first come out.这里的最高级是most没有加the，因为只是说最贵，后面没有名词，没有指示意味，为什么要加the

    对于比较级也是同样的道理，遵循具备指示性可以加，不具备则可用零冠词或不定冠词，这里的指示性是指指示名词，如果连名词都没有不用谈什么the的问题，例如the most expensive，当然这样写法也不算错

    # 专有名词

    # 什么是专有名词？

    即人名、地名一些专有的，即世界上只有一个，一般需要首字母大写，如Bill Gates，与之相反的是普通名词，即这个名词可以用在任何地方，可以重复,是一种通称

    # 专有名词是否可以加限定符？

    不能，他已经表示了唯一性，加限定符多余，如果看到专有名词加了限定符，那么就不是专有名词了

    # 什么是假专有名词？

    即其名词词组的名词部分是一种普通名词,但因其省略了所以看起来像专有名词

    the Danube 表示多瑙河，但其实是省略了river的，完整的是the Danube River,所以这里加了the表示限定，因为river是一种通称，所以我们说一些河流等专有名词需要加the的原因

    the United State of America,需要加the同理，state是一种通称，所以我们说一些国家名需要加the,而其后面的介词短语中的America才是真的专有名词

    the Toms 其实表示的是the Tom's family，所以要加the

    # 什么是专有名词普通化？

    即如果专有名词前面了定冠词或不定冠词或者出现了复数形式，其不再是专有名词

    # 什么是普通名词专有化？

    第一种形式，主语和补语对等，即主语部分是专有名词，那么补语部分可以是普通名词当做专有名词
    Bill Gates is Chairman of Microsoft,因为其主语是专有名词，导致其补语的普通名词也具有唯一性，所以当做专有名词，以下同位语，宾补一样的道理
    第二种形式，同位语对等
    Ban Ki-moon, Secretary General of the UN, is a very powerful man
    第三种形式，宾语与其补语对等
    Queen Elizabeth made Charles Prince of Wales.伊丽莎白女王封查尔斯为韦尔斯亲王。

    # 普通名词抽象化是什么意思？

    即一些普通名词前面没有加限定符，因为其当做了抽象名词，例如by phone表示打电话，on foot表示徒步
