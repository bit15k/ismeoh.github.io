                            # 名词词组

                            # 基本形式

                            # 限定符+形容词+名词，所有名词一律按名词词组看

                            # 三个位置

                            # 限定符

                            • 有哪些类型

                              • 前
                              • 后
                              • 中

                            • 特殊的限定符"冠词"

                              • 第一种，零冠词

                                • 零冠词是虚拟词以便于区分定和不定冠词
                                • 零冠词可以适用不可数名词

                              • 第二种，不定冠词

                                • 用“一个one”来判定是否使用不定冠词

                              • 第三种，定冠词

                                • 1.判断，定冠词可用哪个these/those来判定是否使用
                                • 2.最高级，最高级中的定冠词根据上面"/those/these"来确定是否使用定冠词

                            • 不同的限定符对应名词数目

                              • 注意1，名词的“数”与限定符保持一致
                              • 注意2，修饰可数或把不可数限定符不能混用

                            # 形容词

                            • 注意1，可用名词代替
                            • 注意2，可用名词词组代替，但需转换成形容词，即加连字符同时变为单数，即复合名词，参见复合名词部分
                            • 注意3，可出现副词修饰
                            • 注意4，形容词可出现多个

                            # 名词

                            • 按数目分类

                              • 可数名词

                                • 分类

                                  • 有具体形状的物体
                                  • 度量衡单位

                                • 零限定符，可数名词可以使用零限定符

                              • 不可数名词

                                • 物质名词

                                  • 点1，是什么，即没有固定形状
                                  • 点2，计算用单位即可
                                  • 点3，限定符问题，采用零冠词或者其他限定符
                                  • 点4，物质名词普通化，如果加了数目类限定符变成了普通名词

                                • 抽象名词

                                  • 点1，是什么，即抽象概念
                                  • 点2，限定符问题，采用零冠词或者其他限定符
                                  • 点3，抽象名词普通化，如果加了数目类限定符变成了普通名词

                                • 动名词

                                  • 即动作或事情
                                  • 采用零冠词或者其他限定符

                                • 专有名词

                            # 省略问题

                            • 注1，除限定符都可省，若一名词前面啥都没，前面限定符是零冠词，形容词被省略
                            • 注2，省略名词前提不影响理解

                            # 特殊的名词词组

                            # 特1，复合名词

                            • 是什么

                              • 即形容词位置是名词

                            • 三种写法

                              • 写法1，两个名词合在一起无空隙
                              • 写法2，两个名词加连字符
                              • 写法3，两个名词分开

                            • 单复数问题

                              • 第一类，插入名词位置单复数问题

                                • 一般用单数，即使是复数名词， 除非不用单数会改变意思

                              • 第二类，整个复合名词的单复数

                                • 第一点，s放在最后一个名词
                                • 第二点，介副词修饰的名词放在介副词上

                            # 特2，限定符或名词位置以名词词组

                            代替构成的名词词组中的名词词组

                            • 注意1，限定符位置的名词词组比如代词所有格
                            • 注意2，名词位置放入名词词组要看做一个整体

                            # 特3，专有名词

                            • 是什么

                              • 即一些人名或地名表示唯一的事物，大写

                            • 是否加限定符

                              • 专有名词不需要加限定符因为其唯一性

                            • 一个特殊类型专有名词

                              • 假专有名词
                                • 情况1，假专有名词省略普通名词加之形容词也是名词，造成所谓需要要加the，其本质仍是普通名词
                                • 情况2，国名，家庭名是假专有名词，需要加the，仔细看其名词部分仍是普通名词

                            • 如何转换成普通名词

                              • 加了限定符即可

                            • 普通名词专有化问题

                              • 情况1，主语是专有，补语可变成专有名词
                              • 情况2，名词是专有，同位语可变专有名词
                              • 情况3，宾语是专有，宾补可变专有名词
